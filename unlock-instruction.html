To complete installation, follow these steps:

1.
Close the application "Parental Control Kroha" and open the application "Settings"
2.
In the application "Settings", go to the section "Profile downloaded" . (If you did not find this section, then follow these steps: "General" -> "VPN & Device management" -> "Parental Control Kroha MDM")
3.
Сlick the button "Install" to start installing the profile
4.
Click the button "Trust" to complete the profile installation
5.
After the profile is installed, click the button "Finish"
6.
Return to the application "Parental Control Kroha"
